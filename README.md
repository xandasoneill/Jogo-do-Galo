JOGO DO GALO

A small TicTacToe project, Jogo do Galo in portuguese.
To play you just need to run the interface.py file, the GUI was made using tkinter, all the text is in portuguese, sorry non-portuguese speakers. 
There is the option to play against an AI (Minimax algorithm), you just need to choose the option "jogar contra o galo".
